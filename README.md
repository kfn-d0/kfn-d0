## Projetos / Ferramentas

| Projeto                          | Descrição                                              | Linguagem Principal |
|----------------------------------|--------------------------------------------------------|---------------------|
| [NetworkAnalysisSuite](https://github.com/kfn-d0/NetworkAnalysisSuite) | Análise avançada de rotas de rede (estilo WinMTR aprimorado) | C# |
| [ispdiag](https://github.com/kfn-d0/ispdiag) | Diagnóstico completo de conectividade em dispositivos Android | Java |
| [ISPing](https://github.com/kfn-d0/ISPing) | Monitor de latência leve para Windows (system tray)    | C# |
| [DDNScover](https://github.com/kfn-d0/DDNScover) | Descoberta passiva rápida de subdomínios              | C# |
| [GrafanaBackup](https://github.com/kfn-d0/GrafanaBackup) | Backup automatizado de dashboards do Grafana         | C# |
| [quickauditlinux](https://github.com/kfn-d0/quickauditlinux) | Auditoria rápida de segurança em VMs Linux           | Python |
| [PORTyscan](https://github.com/kfn-d0/PORTyscan) | Scanner de portas para Windows                        | C# |
| [httpresponsetimer](https://github.com/kfn-d0/httpresponsetimer) | Extensão para Chrome que mede tempo de resposta HTTP | JavaScript |
