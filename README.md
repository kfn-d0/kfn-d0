**Network • Observability • Automação**

Gosto de criar ferramentas práticas para diagnóstico de rede, monitoramento de infraestrutura e automação de processos.  

---

## 🛠️ Tecnologias

- **Linguagens principais**: C# (.NET), Java, Python, JavaScript  
- **Áreas de atuação**:
  - Diagnóstico e análise de rede
  - Monitoramento e observabilidade (Grafana, Zabbix)
  - Scanner de portas e subdomínios
  - Automação de backups e auditoria de sistemas
  - Ferramentas desktop e extensões de navegador
